## 1. selfh.st Newsletter

**Subject:** Locally Uncensored — self-hosted AI chat + image gen in one desktop app

Hi,

I built Locally Uncensored, an open-source desktop app that combines LLM chat (via Ollama) and image/video generation (via ComfyUI) in a single interface. Everything runs locally — no API keys, no cloud, no data leaving your machine.

Built with Tauri v2 + React, so it's lightweight (~15 MB) compared to Electron alternatives. MIT licensed.

Why it fits selfh.st: it's a genuine self-hosted AI workflow — chat and image gen unified, with zero external dependencies once set up.

GitHub: https://github.com/PurpleDoubleD/locally-uncensored

Happy to provide screenshots or answer any questions.
