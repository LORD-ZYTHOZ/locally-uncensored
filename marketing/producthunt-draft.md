## Description (260 chars max)

Locally Uncensored is a free, open-source desktop app that combines LLM chat (Ollama) and image/video generation (ComfyUI) in one interface. Everything runs on your hardware — no API keys, no accounts, no data leaving your machine. Built with Tauri + React.

## Maker Comment (first comment after launch)

Hey Product Hunt! I built Locally Uncensored because I was tired of switching between separate apps for local AI chat and image generation.

The idea is simple: one app that talks to Ollama for LLM chat and ComfyUI for image/video gen. No cloud services, no API costs — just your hardware doing the work.

Tech stack: Tauri v2 + React. The whole app is about 15 MB (thanks to Tauri instead of Electron). MIT licensed, so you can fork and modify however you want.

I'd love feedback on what features matter most to you. What would make this your go-to local AI app?

GitHub: https://github.com/PurpleDoubleD/locally-uncensored

## Key Features

1. **Local LLM Chat** — Connect to Ollama and chat with Llama, Mistral, Phi, Gemma, and dozens of other models. All running on your hardware.

2. **Image & Video Generation** — Built-in ComfyUI integration for Stable Diffusion, Flux, and other generative models. Create images without leaving the app.

3. **Complete Privacy** — Zero cloud dependencies. No accounts, no telemetry, no API keys. Your data never leaves your machine.

4. **Lightweight & Native** — Built with Tauri v2 (~15 MB), not Electron. Fast startup, low memory footprint, cross-platform (macOS, Windows, Linux).

5. **Open Source (MIT)** — Fully open source. Inspect the code, contribute, or fork it for your own use.
